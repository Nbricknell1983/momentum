DECISION TREE / GROWTH PRESCRIPTION ENGINE:
Build a structured decision engine that determines the correct product mix dynamically based on the business state.

Inputs:
- website exists / no / weak
- GBP exists / no / weak
- reviews low / medium / strong
- current leads none / inconsistent / strong
- ads yes / no / poor
- service areas
- market opportunity
- competitive difficulty
- business stage startup / growing / established
- urgency
- affordability / investment appetite

Decision logic:
WEBSITE
- if no website → website required
- if weak website → rebuild or expansion required
- if website exists but does not support service/location intent → structural rebuild or SEO expansion required

GBP
- if no GBP → setup/activation required
- if weak GBP → optimisation required
- if low reviews or weak local signals → review and GBP play required

LEAD FLOW
- if no leads or very inconsistent leads → prioritize Google Ads + core conversion fixes
- if leads exist but organic visibility weak → prioritize SEO + GBP
- if traffic exists but conversion weak → prioritize CRO / website conversion improvements

MARKET OPPORTUNITY
- low difficulty / clear demand → SEO-first opportunity
- higher competition → blended SEO + Google Ads
- high-value services and strong search demand → full-stack strategy

BUSINESS STAGE
- startup → foundations stack
- growing → optimization stack
- established → domination / market capture stack

Output:
- recommended product stack
- priority order
- rationale
- expected impact
- expected timeline
- affordability options
