PROJECT: Momentum
FEATURE: Vapi Voice Agent Layer for Sales, Prospecting, Follow-Up, and Appointment Setting

IMPORTANT CONTEXT

This is the Momentum project.
Momentum is the sales, strategy, communication, and account-growth platform.

Momentum owns:
- lead intelligence
- sales execution
- proposal / strategy workflow
- onboarding transition
- cadence
- communication drafting
- execution queue
- expansion / churn / referral logic
- autopilot policy
- scheduled sweeps
- manager workflows

This is NOT the AI Systems website/runtime project.
Do NOT build tenant website assistant components here.
Do NOT build tenant booking/message/status backend logic here unless needed for Momentum’s own internal calling workflows.

CORE GOAL

Build Vapi into Momentum as the voice layer for sales and communication workflows.

Momentum Vapi should support use cases such as:
- outbound appointment setting
- prospecting calls
- inbound lead capture for Momentum
- follow-up calls after strategies/proposals
- qualification / discovery support
- booking meetings onto calendar flows
- account management follow-up calls
- churn-risk intervention calls
- referral ask calls
- reactivation calls

CRITICAL ARCHITECTURE RULE

Vapi is the voice interface layer.
Momentum remains the logic/orchestration layer.

Responsibilities:

Vapi:
- outbound/inbound voice interactions
- call/web voice interface
- tool calling

Momentum:
- lead context
- sales stage logic
- cadence logic
- communication policy
- NEPQ-style question structure
- objection handling structure
- follow-up logic
- account-growth/referral/churn logic

OpenAI / reasoning layer:
- language/reasoning where needed

OpenCore / orchestration:
- policies
- guardrails
- approvals
- allowed actions

IMPORTANT SALES SAFETY RULE

Do NOT just dump NEPQ or Chris Voss prompts into a raw bot.
Use those frameworks as guarded conversation structures inside Momentum logic.

That means:
- Momentum decides allowed conversational goals
- Momentum decides next questions/prompts
- Momentum decides whether an action is allowed
- Vapi executes the conversation channel
