FEATURE 1 — WEBSITE X-RAY

Triggered by:

[ Run Website X-Ray ]

Purpose:
Visually audit the prospect website.

Steps:
1. Load the prospect website URL.
2. Capture a full homepage screenshot.
3. Capture a mobile viewport screenshot.
4. Attempt to load /sitemap.xml.
5. Extract key SEO signals including:
   - title tag
   - meta description
   - H1 headings
   - heading hierarchy
   - internal links
   - word count
   - navigation labels
6. Detect service keywords and location keywords appearing on the homepage.

Output UI:
Display the homepage screenshot with numbered callouts highlighting issues such as:
- weak service clarity
- missing location signals
- weak call to action
- thin content
- unclear page structure

Include a side panel explaining each issue and the recommended fix.


FEATURE 2 — LIVE SERP VISUALISATION

Triggered by:

[ Visualise Search Results ]

Purpose:
Show where the prospect appears in Google search results.

Steps:
1. Allow keyword input or AI suggested keywords.
2. Capture search results for the keyword.
3. Detect:
   - map pack presence
   - organic ranking presence
   - competitor domains
4. Identify whether the prospect domain appears.

Output UI tabs:

SERP View
Prospect Position
Competitors
Opportunities

SERP View must render a clean search results snapshot suitable for presentations.

Prospect Position must show:
- Maps presence: detected / not detected
- Organic presence: detected / not detected
- Best matching prospect page
- Keyword-page relevance score

Competitors tab must list top ranking businesses.


FEATURE 3 — COMPETITOR GAP ANALYSIS

Triggered by:

[ Analyse Competitor Gap ]

Purpose:
Explain why competitors rank higher.

Steps:
1. Identify top 3 competitors from SERP results.
2. Crawl competitor websites using the same crawl engine.
3. Extract structural signals including:
   - service page count
   - location page count
   - average content depth
   - heading structure
   - internal linking signals
   - review signals if visible

Output UI:

Comparison table:

Signal | Prospect | Competitor Average

Service Pages
Location Pages
Content Depth
Internal Linking
Review Signals

Also generate visual comparison charts using Chart.js.


FEATURE 4 — TRAFFIC & REVENUE FORECAST

Triggered by:

[ Forecast Traffic & Revenue ]

Purpose:
Convert SEO opportunity into business impact.

Steps:
1. Use detected keywords and search demand.
2. Estimate current visibility capture.
3. Estimate improved capture after optimisation.
4. Apply conversion rate estimates.
5. Apply job value estimates (or industry averages).

Output metrics:

Estimated Traffic Opportunity
Estimated Lead Opportunity
Estimated Revenue Opportunity

Display charts for:
- traffic growth forecast
- visibility improvement
- revenue opportunity

All forecasts must be labelled as estimates.
