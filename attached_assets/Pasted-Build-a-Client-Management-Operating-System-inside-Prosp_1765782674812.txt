Build a Client Management Operating System inside Prospectr that unifies Pipeline + Clients + Daily Planner, powered by a single Action Queue, Route Planner, Client Health system, Delivery tracking, Client Strategy engine, and AI-driven Next Best Action (NBA) logic.

The system must work across three separate pages but remain fully connected through shared database records so execution, logging, and dashboards stay accurate after refresh.

Create three separate sections:
